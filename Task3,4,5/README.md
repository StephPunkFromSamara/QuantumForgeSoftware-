# QuantumForgeSoftware Knowledge Base

## Описание

Этот проект представляет собой векторный индекс текстов вселенной Гарри Поттера, анонимизированный с помощью словаря замен имен и мест. Индекс создан с использованием ChromaDB для хранения эмбеддингов и последующего семантического поиска.

## Модель

- Модель для генерации эмбеддингов: **sentence-transformers/all-MiniLM-L6-v2**
- Размерность эмбеддингов: 384

## База знаний

- Хранение: **ChromaDB**, папка `chroma_db`
- Коллекция: **knowledge_base**
- Количество чанков в индексе: **13,896**
- Метаданные каждого чанка: `source_file`, `title`, `chunk_id`, `start_pos`, `end_pos`, `word_count`

## Время генерации

- Генерация и загрузка всех эмбеддингов заняла несколько минут (зависит от мощности машины).
